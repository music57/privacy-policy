減肥健康 隱私政策

最後更新日期：2026 年 3 月

一、資料儲存

本應用程式的所有個人資料（包含體重、飲食紀錄、運動紀錄、冰箱食材等）皆儲存於您的裝置本機，不會上傳至任何伺服器。

您的 API Key 使用 Android Keystore 安全儲存，不會以明文形式存放。

二、AI 服務

本應用程式使用您提供的 API Key 連接第三方 AI 服務（OpenAI、Anthropic、Google Gemini）進行飲食分析和營養建議。

當您使用 AI 功能時，以下資料會傳送至您選擇的 AI 供應商：

  • 飲食照片（用於營養分析）
  • 冰箱食材照片（用於食材辨識）
  • 基本身體數據（身高、體重、熱量目標，用於個人化建議）

這些資料的處理受各 AI 供應商的隱私政策規範。

三、開發者不收集資料

本應用程式的開發者不收集、儲存或處理任何使用者的個人資料。

  • 我們不使用任何分析工具或追蹤器
  • 我們不會與第三方分享您的資料
  • 我們不會在裝置外儲存任何使用者資料

四、照片處理

您在應用程式中拍攝的照片會儲存在裝置本機。照片僅在您主動使用 AI 分析功能時才會傳送至 AI 供應商，且不會被開發者存取。

五、資料刪除

您可以隨時透過解除安裝應用程式來刪除所有本機資料。若要撤銷 AI 供應商的 API Key 存取權，請至各供應商的管理介面停用該金鑰。

六、聯絡方式

如有任何隱私相關問題，請透過 Google Play 商店的開發者聯絡方式與我們聯繫。
